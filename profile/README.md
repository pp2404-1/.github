# PollFlow

# Платформа для быстрого создания онлайн-опросников с отображением результатов в реальном времени. Создавайте опросы, делитесь ссылкой и наблюдайте за статистикой в live-режиме!

# Почему PollFlow уникален?
Мы отказались от всего лишнего, чтобы вы могли создать опрос за минуту. Наш интерфейс сфокусирован только на том, что действительно важно: быстрое создание и живая обратная связь. Мы создаём инструмент, в котором гибкий дизайн сочетается с продуманной логикой, чего не хватает в аналогах, но без перегрузки, свойственной корпоративным системам. Чистый и сфокусированный интерфейс позволяет решить задачу за пару кликов — без лишних настроек и путаницы.

# Что умеет?

### Функциональные возможности:
- **Быстрое создание** — интуитивный конструктор без лишних настроек
- **Таймеры** — ограничивай время опроса, система сама его завершит
- **Анонимность** — голоса анонимны по умолчанию
- **Свои варианты** — участники могут предлагать ответы
- **Live-статистика** — диаграммы обновляются без перезагрузки
- **Прозрачность** — участники видят результаты после голосования
- **Защита от накрутки** — один человек = один голос

### Технические особенности:
- **Адаптивный дизайн** — работает на любом устройстве
- **Real-time обновления** — результаты прилетают мгновенно
- **Надежное хранение** — данные сохраняются безопасно и без потерь
- **Масштабируемая архитектура** — спроектирована для работы с растущим числом пользователей

# Запуск с Docker
PollFlow построен на контейнерной микросервисной архитектуре, что позволяет:

**Для пользователей**:

- **Запуск за 60 секунд** — docker-compose up и готово

- **Автоматические обновления** — интерфейс и логика синхронизируются без перезагрузки

- **Надёжность** — падение одного компонента не ломает всю систему

**Для разработчиков**:

- **Полная изоляция** — каждый сервис в собственном контейнере

- **Гибкая разработка** — можно модифицировать фронтенд, не трогая бэкенд

- **Встроенный мониторинг** — логи всех компонентов доступны одной командой

## Быстрый старт:

**1. Установите Docker и Docker Compose**

**2. Клонируйте репозиторий**:

git clone <ваш-репозиторий>

cd pollflow

**3. Запустите все компоненты**:

docker-compose up -d

**4. Откройте в браузере**:

Frontend: http://localhost:3000

API документация: http://localhost:8000/docs

## Внедрение интеллектуального помощника на базе машинного обучения.

Модуль искусственного интеллекта должен анализировать введенный пользователем заголовок опроса и на его основе предлагать релевантные варианты готовых вопросов, а также типовые формулировки ответов.
